Chat Up

A real-time chat application built with Node.js, Express, MongoDB, and Socket.IO, designed with a clean MVC architecture. Just a fun project to becom a better dev
It supports secure 1-to-1 messaging and event-driven communication over WebSockets.

 Features

 JWT authentication & user management

 Real-time private messaging with Socket.IO

 Secure middleware for chat membership & access control

 Modular monolithic design (services, repositories, middlewares)

 Scalable architecture with Redis (for pub/sub)

 Tech Stack

Backend: Node.js, Express

Database: MongoDB

Real-time: Socket.IO

Cache & Scaling: Redis

Auth: JWT + Argon2 password hashing
